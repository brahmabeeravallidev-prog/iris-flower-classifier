# 🌸 Iris Flower Classification

A beginner Machine Learning project to classify iris flowers
into 3 species using Python and scikit-learn.

## 📁 Project Structure
iris-classifier/
├── iris_classifier.py   # Main ML code
├── README.md            # This file
├── iris_pairplot.png    # Feature pair plot
├── iris_boxplot.png     # Box plot chart
├── iris_confusion_matrix.png  # Confusion matrix
└── iris_knn_k_values.png      # KNN k-value chart

## 🛠️ Libraries Used
- numpy
- pandas
- matplotlib
- seaborn
- scikit-learn

## ▶️ How to Run

Install libraries:
pip install numpy pandas matplotlib seaborn scikit-learn

Run the project:
python iris_classifier.py

## 📊 Models Used
- K-Nearest Neighbors (KNN)
- Decision Tree Classifier

## 📈 Results
- KNN Accuracy  : ~100%
- Decision Tree : ~100%
